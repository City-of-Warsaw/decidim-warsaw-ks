# Decidim::StudyNotes

Component for submitting resident comments on the general plan.

## Usage

StudyNotes will be available as a Component for a Participatory Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-study_notes"
```

And then execute:

```bash
bundle
```
