# Decidim::AdUsersSpace

Space dedicated fot users with AD permissions.

## Usage

AdUsersSpace will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-ad_users_space"
```

And then execute:

```bash
bundle
```
