# Decidim::CustomProposals

Component for comments for document.

## Usage

CustomProposals will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-custom_proposals"
```

And then execute:

```bash
bundle
```
