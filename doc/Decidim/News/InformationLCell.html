Overview

This cell is used a base for all List cards. It holds the basic layout so other cells only have to customize a few methods or overwrite views.

def description
  attribute = resource.try(:short_description) || resource.try(:body) || resource.description
  text = translated_attribute(attribute)

  decidim_sanitize_editor(html_truncate(text, length: description_length), strip_tags: true)
end
