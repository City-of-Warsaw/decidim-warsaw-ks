Overview

This class holds the logic to present a `Decidim::Area` for the `AdminLog` log.

Usage should be automatic and you shouldn't need to call this class directly, but here's an example:

action_log = Decidim::ActionLog.last
view_helpers # => this comes from the views
AreaPresenter.new(action_log, view_helpers).present
