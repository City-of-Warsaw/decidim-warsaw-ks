Overview

This Command is used EXCLUSIVELY by Unregistered users, to save statistics data fot the second step of adding new User Question It uses update method, as data is additional to the required body part of the model This command is not used in any other moment

Attributes that are saved through this command:

  • signature

  • email

  • district_id

  • age

  • gender

In this case Boolean field 'edited' has default value of false, as this Command is used only in one specific moment: right after creation to gather statistical data
