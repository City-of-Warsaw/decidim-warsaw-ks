# Decidim::CustomAi

The AI module for handling high volumes of resident feedback submitted via survey, along with administrative tools.

## Usage

CustomAi will be available as a Component for a Participatory Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-custom_ai"
```

And then execute:

```bash
bundle
```
