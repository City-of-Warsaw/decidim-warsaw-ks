# Decidim::ExpertQuestions

The expert question is a participatory process component that allows users to ask designated experts questions

## Usage

ExpertQuestions will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-expert_questions"
```

And then execute:

```bash
bundle
```
