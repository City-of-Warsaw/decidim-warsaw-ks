# Decidim::ExpertQuestions

Component for posting questions for experts in given participatory space.

## Usage

ExpertQuestions will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-expert_questions"
```

And then execute:

```bash
bundle
```
