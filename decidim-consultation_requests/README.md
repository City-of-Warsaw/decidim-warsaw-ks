# Decidim::ConsultationRequests

Consultation request is new type of content to submit requests for public consultations by residents

## Usage

ConsultationRequests will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-consultation_requests"
```

And then execute:

```bash
bundle
```
