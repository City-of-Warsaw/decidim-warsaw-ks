# Decidim::Remarks

Adds new participatory process component that allows users to send remarks.

## Usage

Remarks will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-remarks"
```

And then execute:

```bash
bundle
```
