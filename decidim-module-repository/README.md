# Decidim::Repository
Repository adds the ability media and files management by administarators.

## Usage

Repository will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-repository"
```

And then execute:

```bash
bundle
```
